# Tips-n-Tricks
## Latex 
### useful packages
package `siunitx` for typesetting values with correct units
in options: `[per-mode=symbol]`: / instead of ^-1
### useful commands
Scaling image to the width of the paper: 
 ```
 \begin{figure}
        \centerline{\includegraphics[width=\paperwidth]{figure.pdf}}
 \end{figure}
 ```
